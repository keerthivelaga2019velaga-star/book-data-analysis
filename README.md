# 📚 Book Data Analysis

This project analyzes book information (title, price, rating, and URL) using Python in Google Colab.  
It demonstrates a simple data pipeline: scraping → cleaning → visualization → database storage.

---

## 🚀 Getting Started
1. Open the notebook directly in Google Colab:  
   [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/keerthivelaga2019velaga-star/book-data-analysis/blob/main/BookDataAnalysis.ipynb)

2. Run the cells step‑by‑step to reproduce the analysis.

## 📊 Features
- Web scraping of book data

- Storage in SQLite database

- Python 3.x  
- Jupyter/Colab  
- Libraries: `pandas`, `matplotlib`, `seaborn`, `sqlite3`
